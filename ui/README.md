# PassCheck React UI

## Configure API
Create `.env` in this folder or set an env var at your host:
```
VITE_API_BASE=https://passcheck-whck.onrender.com
```

## Render (Static Site)
- Root Directory: `ui`
- Build Command: `npm ci && npm run build`
- Publish Directory: `dist`
- Env: `VITE_API_BASE` as above
